Computer Organization / Pipelining

A data hazard in instruction pipelining occurs when:

Computer Organization Pipelining Pipeline Hazards
MCQ2026Practice

Choose an Option

Answer

The next instruction depends on the result of a previous instruction

Theory

Data Hazard

Data hazards arise when instructions depend on the results of earlier instructions still in the pipeline.

Solution

  • Instruction I2 may need data produced by I1.
  • If I1 has not completed, I2 cannot proceed safely.
  • This dependency creates a data hazard.
  • Forwarding and stalling are common solutions.
