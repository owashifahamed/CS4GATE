Operating System / Process Management

Which process state indicates that a process is waiting for an event such as I/O completion?

Operating System Process Management Process States
MCQ2026Practice

Choose an Option

Answer

Blocked

Theory

Process States

A blocked process cannot execute until a specific event occurs.

Solution

  • Running means CPU is executing the process.
  • Ready means waiting for CPU allocation.
  • Blocked means waiting for an event such as I/O completion.
  • Therefore the answer is Blocked.
