A 4-to-1 multiplexer requires how many select lines?

Digital Electronics Combinational Circuits Multiplexers
MCQ2026Practice

Choose an Option

Answer

2

Theory

Multiplexer

A multiplexer with n input lines requires log2(n) select lines.

Solution

  • Number of inputs = 4.
  • Select lines = log2(4).
  • = 2.
  • Therefore a 4-to-1 MUX requires 2 select lines.
