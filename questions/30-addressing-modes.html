In which addressing mode is the operand explicitly specified within the instruction itself?

Computer Organization Instruction Set Architecture Addressing Modes
MCQ2026Practice

Choose an Option

Answer

Immediate Addressing

Theory

Immediate Addressing

The actual operand value is part of the instruction.

Solution

  • Example: MOV R1, #10
  • Here 10 is directly present in the instruction.
  • No memory lookup is required for the operand.
  • Hence this is Immediate Addressing.
