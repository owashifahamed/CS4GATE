Theory of Computation / Pushdown Automata

Which of the following language is accepted by a Deterministic Pushdown Automata?

Theory of Computation Pushdown Automata DPDA
MCQ2025PYQ

Choose an Option

Answer

Any regular language.

Theory

Difference Between DPDA and NPDA

L(DFA) = L(NFA) ⊂ L(DPDA) ⊂ L(NPDA) ⊂ L(LBA) ⊂ L(TM)

  • Regular languages are accepted by DFA, NFA, DPDA, NPDA, LBA and TM.
  • Deterministic context-free languages are accepted by DPDA, NPDA, LBA and TM.
  • Context-free languages are accepted by NPDA, LBA and TM.

Solution

  • Option A: Any regular language. Correct. Regular languages are accepted by DPDA.
  • Option B: Any context free language. Wrong. DPDA accepts only deterministic context-free languages.
  • Option C: Wrong. NPDA accepts more languages than DPDA.
  • Option D: Wrong. DPDA does not accept every decidable language.
