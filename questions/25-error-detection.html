Which error detection technique is widely used in Ethernet frames?

Computer Networks Data Link Layer Error Detection
MCQ2026Practice

Choose an Option

Answer

CRC

Theory

Cyclic Redundancy Check (CRC)

CRC is a powerful error-detection mechanism commonly used in network protocols.

Solution

  • CRC treats data as a binary polynomial.
  • The sender appends a remainder.
  • The receiver performs the same division.
  • A non-zero remainder indicates an error.
  • Hence CRC is correct.
