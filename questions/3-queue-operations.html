If elements 10, 20, 30 and 40 are inserted into an empty queue and one deletion operation is performed, which element will be deleted?

Data Structures Queues Queue Operations
MCQ2026Practice

Choose an Option

Answer

10

Theory

Queue

A queue follows the FIFO (First In First Out) principle. The first element inserted is the first one removed.

Solution

  • Elements are inserted in the order: 10, 20, 30, 40.
  • Queue deletion removes the element at the front.
  • The first inserted element is 10.
  • Therefore, 10 will be deleted.
