For a square matrix A, if det(A) = 0, then A is:

Engineering Mathematics Linear Algebra Matrices
MCQ2026Practice

Choose an Option

Answer

Singular

Theory

Determinants

A matrix is singular if its determinant is zero, implying that its inverse does not exist.

Solution

  • det(A)=0.
  • Inverse exists only when det(A) `∠0.
  • Therefore A is non-invertible.
  • Such a matrix is called Singular.
