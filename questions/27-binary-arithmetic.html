What is the decimal equivalent of the binary number 101101?

Computer Organization Number Systems Binary Arithmetic
MCQ2026Practice

Choose an Option

Answer

45

Theory

Binary to Decimal Conversion

Each binary digit represents a power of 2 based on its position.

Solution

  • 101101_2= 1×2^5+ 0×2^4+ 1×2³ + 1×2² + 0×2¹ + 1×2^0
  • = 32 + 0 + 8 + 4 + 0 + 1
  • = 45
  • Therefore the decimal equivalent is 45.
