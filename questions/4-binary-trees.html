A full binary tree contains 15 nodes. How many leaf nodes does it have?

Data Structures Trees Binary Trees
MCQ2026Practice

Choose an Option

Answer

8

Theory

Full Binary Tree

In a full binary tree, every node has either 0 or 2 children. The number of leaf nodes can be calculated using the relation: Leaf Nodes = (Total Nodes + 1) / 2.

Solution

  • Total nodes = 15.
  • Leaf Nodes = (15 + 1) / 2.
  • = 16 / 2.
  • = 8.
  • Therefore, the number of leaf nodes is 8.
