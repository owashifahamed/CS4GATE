Choose an Option

Answer

Theory

Lexical Analysis

The lexical analyzer scans the source program and converts character sequences into meaningful tokens.

Solution

  • Lexical analyzer is the first phase of a compiler.
  • It groups characters into lexemes.
  • Each lexeme is converted into a token.
  • Therefore the output is Tokens.
