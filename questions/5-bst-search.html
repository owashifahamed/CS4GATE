What is the worst-case time complexity of searching for an element in a Binary Search Tree (BST) containing n nodes?

Data Structures Binary Search Tree BST Search
MCQ2026Practice

Choose an Option

Answer

O(n)

Theory

Binary Search Tree (BST)

A Binary Search Tree allows efficient searching when balanced. However, if the tree becomes skewed, its height can become n.

Solution

  • In a balanced BST, search complexity is O(log n).
  • In the worst case, the BST becomes skewed like a linked list.
  • The height of the tree becomes n.
  • All nodes may need to be visited.
  • Therefore, the worst-case complexity is O(n).
