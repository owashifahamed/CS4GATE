Which optimization technique replaces repeated computation of the same expression by a single computed value?

MCQ2026Practice

Choose an Option

Answer

Common Subexpression Elimination

Theory

Common Subexpression Elimination

If an expression is evaluated multiple times and operands do not change, the result can be reused.

Solution

  • Example: a+b appears multiple times.
  • Compute it once and store the result.
  • Reuse the stored value later.
  • This reduces computation cost.
