Computer Networks / OSI Model

Which layer of the OSI model is responsible for end-to-end process communication?

Computer Networks OSI Model Network Layers
MCQ2026Practice

Choose an Option

Answer

Transport Layer

Theory

Transport Layer

The Transport Layer provides end-to-end communication services, segmentation, flow control and error control.

Solution

  • Network Layer handles routing.
  • Data Link Layer handles node-to-node communication.
  • Transport Layer provides end-to-end process communication.
  • Hence the correct answer is Transport Layer.
