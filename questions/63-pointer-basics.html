What will be the value of x after executing the following code? int x = 10; int *p = &x; *p = 20;

Programming & Data Structures in C Pointers Pointer Basics
MCQ2026Practice

Choose an Option

Answer

20

Theory

Pointers

A pointer stores the memory address of another variable. Dereferencing a pointer allows modification of the variable it points to.

Solution

  • x is initialized to 10.
  • p stores the address of x.
  • *p = 20 updates the value at that address.
  • Therefore x becomes 20.
