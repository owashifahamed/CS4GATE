What is the size of the char data type in C?

Programming & Data Structures in C C Basics Variables and Data Types
MCQ2026Practice

Choose an Option

Answer

1 byte

Theory

Character Data Type

In C, a char occupies exactly 1 byte of memory and is used to store a single character.

Solution

  • The C standard defines sizeof(char) as 1 byte.
  • It stores a single character such as 'A' or 'b'.
  • Therefore the size of char is 1 byte.
