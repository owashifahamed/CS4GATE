DBMS / Normalization

Which normal form removes partial dependency from a relation?

DBMS Normalization Functional Dependency
MCQ2026Practice

Choose an Option

Answer

2NF

Theory

Second Normal Form (2NF)

A relation is in 2NF if it is in 1NF and every non-prime attribute is fully functionally dependent on the entire candidate key.

Solution

  • 1NF removes repeating groups.
  • 2NF removes partial dependencies.
  • 3NF removes transitive dependencies.
  • Hence the correct answer is 2NF.
