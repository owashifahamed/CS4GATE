The Activity Selection Problem is optimally solved using which design technique?

Algorithms Greedy Algorithms Activity Selection
MCQ2026Practice

Choose an Option

Answer

Greedy Method

Theory

Greedy Algorithms

The Activity Selection Problem chooses the maximum number of non-overlapping activities by repeatedly selecting the activity that finishes earliest.

Solution

  • Sort activities according to finishing times.
  • Select the earliest finishing activity.
  • Choose the next compatible activity.
  • This greedy choice leads to an optimal solution.
