A stack follows which of the following principles?

Data Structures Stacks Stack Operations
MCQ2026Practice

Choose an Option

Answer

LIFO

Theory

Stack

A stack is a linear data structure where insertion and deletion take place at the same end called TOP.

Solution

  • Stack insertion is PUSH.
  • Deletion is POP.
  • The last inserted element is removed first.
  • Hence Stack follows Last In First Out (LIFO).
